# 抖音私信 AI 自动回复助手

> 一个运行在浏览器里的抖音网页版私信自动回复扩展,接入 DeepSeek AI,能读懂对方消息、按人格自定义回复、记住每个人的聊天历史、还能反驳喷子。

**⚠️ 仅供学习与技术研究使用。使用自动化工具操作抖音可能违反其用户协议,存在账号风险,请自行承担后果。严禁用于营销骚扰、刷量等违规用途。**

---

## ✨ 功能

- 🤖 **AI 智能回复** — 接入 DeepSeek,读懂对方消息内容后自然回复(不是死板的固定话术)
- 🧠 **分人记忆** — 每个人的聊天历史单独记住,回复能接上前文
- 🎭 **自定义人格** — 自己写提示词,让 AI 按你的风格/人设说话
- 🛡️ **喷子反驳** — 自动识别挑衅辱骂,机智怼回去
- ✅ **白名单** — 可设只回复指定的人
- 🚫 **智能分类** — 自动区分 广告/喷子/骗子/真实用户,分别处理
- 👤 **拟人化操作** — 模拟真人打字节奏和点击,降低风控风险
- 💾 **记忆备份** — 一键导出/导入,换电脑不丢数据
- 📦 **纯本地运行** — 无服务器,不上传任何数据,API Key 和记忆只存你本地浏览器

## 📥 安装(小白看这里)

> 本扩展未上架 Chrome 商店,需要用「开发者模式」手动加载。跟着做即可,3 分钟搞定。

### 第一步:下载

点本页右上角绿色 **Code** 按钮 → **Download ZIP**,或直接下载仓库里的 `dy-reply-extension.zip`。

### 第二步:打开浏览器「开发者模式」(关键!)

**Chrome 浏览器:**
1. 地址栏输入 `chrome://extensions/` 回车
2. 打开**右上角**的 **「开发者模式」** 开关

**Edge 浏览器:**
1. 地址栏输入 `edge://extensions/` 回车
2. 打开**左下角**的 **「开发人员模式」** 开关

```
┌─────────────────────────────────────────────┐
│  扩展程序                    开发者模式 [●━]  │ ← 先打开这个开关
├─────────────────────────────────────────────┤
│  [加载已解压的扩展程序] ...                   │ ← 开启后才会出现
└─────────────────────────────────────────────┘
```

### 第三步:把 zip 拖进去安装

**开发者模式打开后**,直接把下载的 `dy-reply-extension.zip` **拖到 `chrome://extensions/` 页面上**松手,即可安装。

> 💡 如果拖 zip 不成功,就先把 zip **解压成文件夹**,再点 **「加载已解压的扩展程序」** 选那个文件夹。

安装成功后,浏览器右上角会出现一个**紫色 AI 图标**。找不到就点拼图 🧩 图标,把它固定 📌。

## 🚀 使用

### 1. 申请 DeepSeek API Key(AI 大脑)
- 打开 https://platform.deepseek.com/ 注册登录
- 充值(1 元起,够用很久)
- 「API keys」→ 创建 → 复制以 `sk-` 开头的 Key

### 2. 开始用
1. 打开并登录抖音网页版 https://www.douyin.com
2. 进入**私信**页面
3. 页面**右下角**出现悬浮窗「抖音自动回复」
4. 粘贴你的 **DeepSeek API Key** → 点「💾 保存 Key」
5. (可选)在「AI 人格/提示词」写你的人设
6. 点 **「启用自动回复」**
7. 完成!有人发私信会自动 AI 回复

### 功能速查
| 功能 | 操作 |
|---|---|
| 收起面板 | 点右上角「—」缩成小圆球,点圆球展开 |
| 只回指定人 | 勾「只回复勾选的人」→ 打开某人对话 → 点「读取当前对话昵称」→ 勾选 |
| 自定义风格 | 在「AI 人格/提示词」里写,如"你是毒舌可爱的博主" |
| 备份记忆 | 「⬇️ 导出」存文件;换电脑后「⬆️ 导入」 |

## ❓ 常见问题

**装的时候提示"扩展程序已损坏"?**
路径别带中文、别太长。解压到 `D:\tool\` 这种简单路径,移除后重新加载。

**右下角没悬浮窗?**
① 确认在 `www.douyin.com` 且已登录 ② 按 `Ctrl+F5` 刷新 ③ 确认插件已启用。

**AI 不回复?**
① 检查 DeepSeek Key 填了没、充值了没 ② 确认点了「启用自动回复」。

**用着用着突然不能用了?**
抖音改版可能导致页面结构变化、扩展失效。这是此类工具的固有特性,需要更新代码里的选择器适配。欢迎提 Issue 或 PR。

## 🛠️ 开发者

改完代码后重新打包:
```bash
python build.py   # 生成 dy-reply-extension.zip
```

技术栈:纯原生 JS 的 Chrome MV3 扩展。核心逻辑在 `content.js`。
- DOM 选择器基于抖音网页版当前结构,抖音改版后可能需要更新。
- AI 走 DeepSeek OpenAI 兼容接口(`fetch` 直连,无中间服务器)。
